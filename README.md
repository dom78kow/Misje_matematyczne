# Misja 01 -- Baza Mars Alfa

W roku 2048 szkolna ekspedycja ląduje na Marsie. Awaria komputera
pokładowego sprawia, że załoga musi samodzielnie podejmować decyzje
niezbędne do uruchomienia Bazy Mars Alfa. Uczestnik misji analizuje
dokumenty, wykonuje obliczenia i przygotowuje raport dla Centrum
Dowodzenia.

Materiały Misji 01 znajdują się w katalogu `Misja01_Baza_Mars_Alfa`.
Najpierw przeczytaj `Misja01_Baza_Mars_Alfa/Wprowadzenie.md`, a następnie
realizuj kolejne rozdziały `Misja01_Baza_Mars_Alfa/Scenariusz Misji 01.md`.
Dokumenty potrzebne do wykonania zadań znajdują się w katalogu
`Misja01_Baza_Mars_Alfa/documents`.
